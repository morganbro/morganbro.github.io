UX Designer at Dagbladet

As of 2015, my primary job has been as a UX designer for Dagbladet.no. In my period in Dagbladet, I've redesigned its frontpage, made a new responsive header for the web site and iOS/Android apps and created breaking news article components for the editorial staff. Further down I've sampled some of the projects I'm proud of!

www.dagbladet.no

The new Dagbladet header (2019).

The old header was not fully responsive and relied on a mobile version for smaller screens. The new one is fully responsive and is now more accessible for screen readers. If a user has Dagbladet pluss membership, they will now see their name in the header, and can now toggle a dropdown to access their account page or log out. The header is also smaller, stickier, lighter and re-usable for other Aller Media sites.
